#CyberChat - Real-Time Web Chat Application
Experience seamless, live communication with CyberChat, a lightweight and efficient chat application deployed on a free hosting platform.

to see details: Visit cyberchat.unaux.com

 Features

Real-Time Messaging: Instant message delivery powered by WebSockets.

easy to use it uses only your name and your room code (if you need to create one or join existing room).

Clean & Responsive UI: A minimalist interface that works on both desktop and mobile devices.


 Built With

Frontend: HTML, CSS, JavaScript (ES6+)

Backend: php
