#CyberChat - Real-Time Web Chat Application
Experience seamless, live communication with CyberChat, a lightweight and efficient chat application deployed on a free hosting platform.

to see details: Visit (cyberchat.unaux.com)

*Features

-Real-Time Messaging: Instant message delivery

-easy to use it uses only your name and your room code (if you need to create one or join existing room).

-you change your room into public or private (if your room is private no one see your chats with others "only invited users see ")

-if you join a room one time you rejoin in next time no need to remember room code 

-you send and recieve images,videos ,audios and some file extensions inside a chat with fast load speed

-Clean & Responsive UI: A minimalist interface that works on both desktop and mobile devices.

-if you need to use this in mobile app download from github


*Built With

-Frontend: HTML, CSS, JavaScript (ES6+)

-Backend: php
